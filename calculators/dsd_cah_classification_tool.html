DSD + CAH Diagnostic Navigator

A glassbox decision-support tool for the endocrine fellow evaluating a Difference/Disorder of Sex Development. Enter history, structured examination (External Masculinisation Score + Prader stage), karyotype/imaging and the LC-MS/MS steroid panel; the model ranks the differential with a fully transparent weighted score, tells you which single test would most change the ranking, and holds you to a completeness checklist so nothing is missed. Educational aid — not a diagnosis.

      2History score

      Family, maternal and antenatal history reweight the differential before a single hormone is drawn — consanguinity and a prior affected sib point to recessive enzyme defects; maternal virilisation points to an aromatase/placental or maternal-source cause.

      3Examination score

      Build the External Masculinisation Score (Ahmed et al., range 0–12; lower = more undervirilised) and the Prader stage (1–5 for a virilised 46,XX). A palpable gonad is the single most decisive bedside sign — a gonad in the labioscrotal fold is a testis/ovotestis and effectively excludes 46,XX CAH.

      External Genitalia Score (EGS) — granular 5-landmark scale

      A finer-grained virilisation score (0–12). Typical females score 0; median for a full-term male is 12 (slightly lower in preterms). EGS < 9 in an infant expected to be phenotypically male (e.g. severe hypospadias) prioritises endocrine/genetic screening; EGS ≤ 4 in highly atypical genitalia statistically correlates with a female sex-assignment pathway — a decision that always belongs to the MDT, never to a score alone.

      4Karyotype, imaging & genetics

      Karyotype is the top-level classifier (sex-chromosome vs 46,XY vs 46,XX DSD). Uterus on pelvic ultrasound, gonadal position/type, and targeted genetics route the rest.

      5Steroid / CAH / gonadal panel

      Enter numeric value + reference interval for auto Low/Normal/High, or set a manual status when using age/sex/stage-specific cut-offs. LC-MS/MS profiling (17-OHP, 21-deoxycortisol, 11-deoxycortisol, DOC, the 11-oxygenated and backdoor markers) is the modern standard (Claahsen-van der Grinten 2022; Speiser 2018).

      6Differential — glassbox ranking

      Score = Σ (weight × fired feature). Positive terms support the diagnosis; negative terms are contradictions that pull it down. Match % = max(0, score) ÷ total positive weight. Data % = fraction of that diagnosis's discriminators you have actually entered. Expand any card to audit every term.

      7Next best test — information gain

      For every input you have not yet entered, the model sums the weight it carries across the current leading diagnoses. The higher the number, the more that single result would move the ranking — this is how you converge without ordering everything blindly.

      8Suspect a diagnosis → targeted work-up

      Working the other way round: pick a diagnosis you already suspect (or a whole group) and the tool lists exactly which tests confirm or refute it — ranked by how much weight each carries in the model, flagged against what you have already entered. Choose a group alone to see the tests that best discriminate within that group.
